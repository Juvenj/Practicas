WOKWI Electrónica

Resumen

Este proyecto consiste en un sistema que detecta si hay un perro o un gato mediante inteligencia artificial. Usamos el modelo YOLO para realizar la detección en tiempo real a través de una cámara web conectada a una Raspberry Pi. La Pi procesa la imagen y envía el resultado a un ESP32, que se encarga de controlar una luz LED y un buzzer. Si se detecta un perro, se enciende un LED azul y se activa un sonido específico; si se detecta un gato, se prende un LED rojo con una frecuencia distinta en el buzzer. De esta forma, el sistema combina visión por computadora y microcontroladores para reconocer mascotas de forma sencilla y eficiente.

Introducción

En este proyecto se desarrolla un sistema inteligente de reconocimiento de mascotas que utiliza inteligencia artificial para detectar si el objeto en escena es un perro o un gato. El objetivo principal es identificar visualmente a la mascota y, en función del resultado, activar una luz LED de color específico (azul para perro, rojo para gato) y emitir un sonido diferente para cada uno mediante un buzzer. Para la detección se utilizó YOLO (You Only Look Once), un modelo de visión por computadora ampliamente usado por su eficiencia y velocidad en tareas de detección de objetos en tiempo real. La arquitectura del sistema combina una Raspberry Pi, que corre el modelo de YOLO y procesa las imágenes captadas por una cámara web, junto con un ESP32, el cual actúa como microcontrolador para controlar los LEDs y el buzzer. La Raspberry Pi se encarga del procesamiento pesado: recibe las imágenes desde la cámara, ejecuta la detección con YOLO, y envía los resultados al ESP32. Este, a su vez, interpreta la información y activa los componentes físicos correspondientes: una luz LED azul y un tono específico si se detecta un perro, o una luz roja y un tono distinto si se trata de un gato.

Conclusiones

En este proyecto logramos implementar un sistema de detección de mascotas utilizando inteligencia artificial con YOLO, controlando luces y sonidos según el tipo de animal detectado. Además, se logró enviar la información a una página web a través de la misma red, mostrando en tiempo real cuándo se detecta una mascota. Esto demuestra cómo se puede integrar visión por computadora, microcontroladores y plataformas web de forma efectiva.
