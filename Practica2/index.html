Resumen

Se configuró el entorno de simulación en WOKWI o Arduino IDE y se probaron tres sensores. El PIR detectó movimiento y activó una respuesta, el potenciómetro ajustó la intensidad de un LED o envió datos al monitor serial, y el DHT22 midió temperatura y humedad. Luego, se programó un monitoreo para el sensor PIR basado en la documentación oficial, verificando su sensibilidad y tiempo de activación. Esta práctica permitió comprender el uso de sensores y su implementación en Arduino.

Se agregaran videos del WOKWI para mejor entendimiento de las conexciones

Introducción

En esta práctica, se trabajará con diferentes tipos de sensores en Arduino utilizando el simulador WOKWI o el hardware real. Se explorará el funcionamiento de un sensor digital de presencia (PIR), un sensor analógico (potenciómetro) y un sensor inteligente (DHT22). Cada sensor será probado para comprender su principio de operación y su integración en un sistema de monitoreo. Finalmente, se desarrollará un programa para monitorear un sensor PIR, detectando movimiento y generando una respuesta visual o en el monitor serial. Aprender a usar el ESP32 es importante porque nos permite crear proyectos de automatización, como encender y apagar luces, medir temperatura o incluso hacer dispositivos inteligentes. En esta práctica, conoceremos cómo funcionan sus pines digitales y aprenderemos a controlarlos con programación, lo que nos ayudará a entender mejor cómo hacer que un dispositivo responda a nuestras órdenes.
