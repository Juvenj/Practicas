Resumen

En esta práctica se conectó el ESP32 a ThingSpeak mediante Wi-Fi para enviar datos en tiempo real desde sensores como el botón, potenciómetro, DHT22 y PIR. Se configuró el entorno en Arduino IDE y se implementó la lectura y transmisión de datos, aplicando conceptos básicos de IoT y comunicación en la nube.

Se agregaran videos del WOKWI para mejor entendimiento de las conexciones

Introducción

En el contexto del Internet de las Cosas (IoT), la capacidad de recopilar, transmitir y analizar datos en tiempo real es clave para el desarrollo de soluciones inteligentes. El ESP32, gracias a su conectividad Wi-Fi, se presenta como una herramienta versátil para establecer comunicación entre sensores y plataformas en la nube. ThingSpeak es una de estas plataformas, destacada por su facilidad de uso para almacenar y visualizar datos de manera remota. Esta práctica tiene como objetivo comprender el funcionamiento e integración entre el ESP32 y ThingSpeak. A través del uso de sensores digitales, analógicos e inteligentes como el botón, el potenciómetro, el DHT22 y el PIR, se llevará a cabo la lectura de datos y su transmisión en tiempo real. Mediante este proceso, se busca fortalecer los conocimientos prácticos sobre IoT, ofreciendo una base útil para la creación de sistemas automatizados y soluciones tecnológicas basadas en la nube.

Conclusiones

Al terminar esta práctica, aprendimos a integrar hardware y software en un proyecto de IoT de forma sencilla y práctica. Configurar el ESP32 y conectarlo con ThingSpeak nos ayudó a entender cómo se pueden monitorear datos a distancia y analizarlos en tiempo real. También trabajar con distintos sensores nos permitió ver lo versátil que puede ser el ESP32 en diferentes tipos de aplicaciones. ¿Quieres que suene todavía más relajado, tipo presentación oral, o así está bien para un informe escrito?
