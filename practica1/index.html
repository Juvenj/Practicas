- Resumen -

En esta práctica aprenderás a usar las entradas y salidas digitales del ESP32, configurarás el IDE de Arduino para programar la placa y revisarás los pines disponibles para su uso, luego desarrollarás varios programas para controlar un LED, primero encendiéndolo y apagándolo con código, después haciendo que parpadee con retardos, luego aprenderás a controlarlo con un botón y después con dos botones usando condiciones lógicas OR y AND, esto ayudará a entender cómo funcionan los pines digitales y cómo interactuar con el entorno a través de sensores y botones.

Se agregaran videos del WOKWI para mejor entendimiento de las conexciones

- Introducción -

El ESP32 es un pequeño dispositivo que funciona como un cerebro para controlar diferentes cosas, como luces, motores o sensores. Es muy útil porque tiene Wi-Fi y Bluetooth, lo que permite conectarlo a internet o a otros dispositivos sin necesidad de cables. Aprender a usar el ESP32 es importante porque nos permite crear proyectos de automatización, como encender y apagar luces, medir temperatura o incluso hacer dispositivos inteligentes. En esta práctica, conoceremos cómo funcionan sus pines digitales y aprenderemos a controlarlos con programación, lo que nos ayudará a entender mejor cómo hacer que un dispositivo responda a nuestras órdenes.
